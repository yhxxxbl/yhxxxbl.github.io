刚开始写 女朋友就发来好吃的诱惑我 /难过

关于实习

在纷享销客实习也有一个多月了 这一个月做了一些简单的工作：

  • 给App接入了OPPO的推送SDK
  • 适配了Android8.0 包括通知渠道 未知应用安装权限、悬浮窗权限
  • 优化了一些小问题 比如轮询的回调策略 云控切换SDK配置等等

老大给我的分配的部门是技术架构组也安排跟架构组坐在一起 平时看大佬们用热修复改BUG、设计架构 萌新不由得瑟瑟发抖 也深知应有一颗敬畏之心 更加努力学习
老大安排了一位导师给我，一个呆萌呆萌的栋哥啊哈哈啊哈哈，我也经常问一些十分简单的问题，他也都耐心给我解答
坐在我旁边儿的大佬用Pyhton写的增量编译给我们的项目编译时间从我host一次要二十分钟提高到现在的几十秒，以前看阿里的文章写技术改变世界还嗤之以鼻，现在算是感受到技术的魅力了
开始喜欢这家公司了

关于学习

终于在大学生涯的尾声体会到了知识的力量 明白了学习的重要性
最近看了极客时间上的深入拆解Java虚拟机和Java核心技术36讲
对JVM有了宏观上的认识 一些JVM的知识在实际编码过程中也是很有意义的，比如熟悉类加载机制你就知道哪部分代码块会在什么时间被加载到内存中等等等
这段时间把精力一直放在Java基础上 过段时间要开始着手深入学习Android了
很多之前学习的事件分发机制 View的原理都没有深入的实践
仍然需要深入 持续的学习 加油

关于生活

三个月的时间说长不长说短也不短 这三个月给自己的改变真的很大
刚来北京的时候举目无亲 也没有工作 也不知道自己能不能找到工作
不知道自己坚持的喜欢的Android能不能养的起我，不知道不听别人Android行情不好转方向的劝告是不是正确的 住在四人的脏乱差宿舍里 每天室友抽烟 唠嗑到深夜
那段时间每天八点出门到附近大学上自习 在北京农学院被赶走 在华北电力占座位
那段时光是痛苦的也是宝贵的
现在搬到了霍营 不用坐死亡的607路公交 虽然每天上下班也要三个小时 但是回到家看到自己的小屋也算是得到了一点安慰
生活没有什么苦 我觉得真正苦的是努力过后看不到希望
为了爹妈老婆孩子 努力努力再努力
21岁说这些不知道是不是太早

概念

Git是一个开源的分布式版本控制系统，用于敏捷高效地处理任何或小或大的项目。

Git 是 Linus Torvalds 为了帮助管理 Linux 内核开发而开发的一个开放源码的版本控制软件。

Git 与常用的版本控制工具 CVS, Subversion 等不同，它采用了分布式版本库的方式，不必服务器端软件支持(摘自百度百科)
